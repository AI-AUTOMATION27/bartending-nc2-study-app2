# NC II Bar Operations Quiz

A simple multiple-choice reviewer website for NC II Bar Operations, Wine Service, Champagne Service, and Sensory Evaluation topics.

## What Is Inside

- `index.html` - the main webpage
- `styles.css` - the design and mobile layout
- `app.js` - the questions, answers, scoring, and quiz behavior

## How To Open

Open `index.html` in any browser.

This works on:

- Laptop or desktop browsers
- Android phones
- iPhones
- Tablets

## How To Use

1. Choose an answer for each question.
2. Use the section buttons to jump between topics.
3. Click **Submit Answers** to see the score.
4. Review the highlighted correct and wrong answers.
5. Click **Reset** to answer again.


## Notes

This quiz runs fully in the browser. It does not need a database, Python server, or login.
